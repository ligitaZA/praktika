Events

Events yra visi veiksmai kurie vyksta puslapyje.(pelės paspaudimas, scroll'inimas, klaviatūros paspaudimas, failo įkėlimas ....)
Norėdami stebėti ar įvyksta event , turime pradžioje nurodyti(selectinti su querySelector), kurioje DOM dalyje tas įvykis gali nutikti

Selektinus , kurioje dalyje klausomės įvykio, reikia aprašyti įvykio klausymąsį. Tai padarome rašydami "addEventListener" metodą, kuris priima 2 parametrus : koksĮvykis(paspaudimas pelės, klaviatūros, užėjimas pelės, pažymėjimas), kąDarytiNutikusĮvykiui (vyksta funkcija).
Galima suteiksti event listener tik konkreciam elementui pvz paragrafui, paveiksleliui ir t.t

DOM

Document object model. Document - puslapis, kuriame yra visa vartotojui atvaizduojama informacija(tekstas, forma, paveiksliukas ...)
object - puslapio elementas(html, span, icon, viskas)
Modelis - objektų struktūra kažkuriame elemente (tekstas, peveiksliukas, forma, lentelė...)

Norėdami per JS parinkti kažkurią Dom dalį, turime rašyti tokį kodą : "document.querySelector" arba "document.querySelectorAll". querySelector priima vieną parametra - selektorių " "(tokį, kokį rašėme CSS)
