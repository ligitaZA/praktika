Tag'u sarasas

  • p - paragraph - Paragrafas - Skirtas rašyti dideliam(arba ne) kiekiui tekstinės informacijos, pastraipai
  • h1-h6 - Headings from 1 to 6 - Antraštės pavadinimams nuo 1 iki 6 - Skirtos rašyti pavadinimams, tarpusavyje skiriasi teksto dydžiu
  • img - Image tag'as - Įkelti nuotraukas is interneto arba is local filo - Kartu naudojamas su src ir alt atributais
  • a - Hypertext anchor - Nuoroda, link'as
  • hr - Thematic break(horizontal line) - Sukuria eilutes lužį su linija
  • br - Break line - Eilutės lužis be linijos
  • ul - Unordered list - Nerikiuotas sarašas
  • ol - Ordered list - Rikiuotas sarašas
  • li - List item - Sąrašo elementas rikiuotam arba nerikiuotam sarašui
  • meta - Skirtas nurodyti puslapio meta duomenis
  • title - Skirtas nurodyti puslapio tab'o pavadinimą

Atribute sąrašas

  • lang - Language - Kalba - Naudinga automatiniams skaitytuvams
  • charset - Character set - Simbolių rinkinys, nurodo kokią simbolių biblioteką naudoti
  • src - Source - Šaltinis - Skirtas nurodyti failą
  • alt - Alternative text - Alternatyvus tekstas pav. neužkrovimo atveju
  • height - Nurodo elemento aukštį
  • width - Nurodo elemento plotį
  • href - Hyper link reference - Nuoroda į failą arba puslapį
  • target - Nurodo kur atidaryti nuorodą(naujam puslapyje ar ne)
