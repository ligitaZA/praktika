Paveldėjimas

Paveldėti stilius gali tik vaikinis elementas iš tėvinio elemento.(iš seserinio pvz kaip sitoje sekcijoje tevinis section, h1 seserinis p elementui)
Paveldimi stiliai yra stiliai kurie keičia elemento išvaizdą, nebūseną pvz.: spalva, teksto dydis, teksto stilius, ...
Nepaveldimi stiliai yra tie, kurie keičia elemento būseną. pvz.: plotis, aukštis, display, border, position, ...
Net paveldėti stiliai gali būti neatvaizduoti dėl stilių prioritetų.

Numatytieji stiliai

Numatytieji stiliai yra automatiskai naršyklės suteikti stiliai. Tarp skirtingų naršyklių numatytieji stiliai gali minimalistiškai keistis ir taip sugadinti vaizdą skirtingų naršyklių vartotojams.
Tam kad nebūtų problmemų, kuriant yra naudojami reset arba normalize css'ai.
reset - resetina visus stilius (ištrina)
normalize- suvienodina visus stilius tarp naršyklių ir panaikina dalį kenkėjiškų stilių.
