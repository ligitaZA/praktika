Stilizavimo tag'ai

  • b - Pastorina tekstą
  • strong - Pastorina tekstą (turi semantinę reikšmę, kad tie žodžiai yra kažkuo svarbesni)
  • i - Pasviras tekstas
  • em - emphasize - Pasviras tekstas (turi semantinę reikšmę, yra svarbesnis)
  • big - Padidina tekstą (pasenęs, nenaudojamas)
  • small - Sumažina tekstą
  • sup - Pakelia tekstą į viršų ir šiek tiek jį sumažina
  • sub - Nuleidžia tekstą į apačią ir jį sumažina
  • span - Išskiria kažkokį elementą ar jo dalį
  • center - Centruoti tekstą (senas nenaudojamas, kodas)

Form'ų atributai

  • type - Input tag'o tipp'as

  • id - Unikalus pavadinimas, į kurį nurodo for atributas
  • name - Įvesties lauko vardas, kuris bus perduotas back'ui kartu su tos įvesties reikšme
  • cols - Stulpelių kiekis textarea elemente
  • rows - Eilučių kiekis textarea elemente
  • placeholder - Tekstas, esantis įvesties lauke, pradedant pildyti įvesties lauką- pradingsta
  • value - Įvesties reikšmė, kuris bus perduota į back'ą(serverį)
  • label - Priskirti colgoup'ui pavadinim1
  • selected - Numatytasis select tag'o viduje esantis option'as
  • disabled - Pasirinkus negalima rinktis tos dalies
  • minlength - Trumpiausia galima įvestis
  • maxlength -Ilgiausia galima įvestis
  • min - Mažiausia leistina įvestis
  • max - Maksimali leistina įvestis

  • pattern - Reikalauja, kad įvesties laukelyje būtų nurodytas formatas(formatas nurodomas regex (regular expresion))
  • checked - Numatytasis radio arba checkbox pasirinkimas
