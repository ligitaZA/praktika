Pseudo Selektoriai

  • *
  • element
  • .class
  • #id
  • selector, selector
  • selector selector
  • selector > selector
  • selector.class#id - elementas turitis nurodytą klasę arba klases ir/arba nurodytą #id
  • selector1 + selector2 - iš karto už selector1 einantis selector2
  • selector1 ~ selector2 - už selector1 visi einantys selector2 (tame pačiame lygmenyje)
  • [atribute] - atributas
  • [atribute=_value] - atributas turintis nurodytą reikšmę
  • [atribute~=value] - atributas, kurio reikšmėje yra nurodytas žodis
  • [atribute*=value] - atributas, kurio reikšmėje yra nurodyta žodžio dalis
  • atribute|=value - atributas, kurio reikšmė yra value arba value-kažkas
  • atribute^=value - atributas, kurio reikšmė prasideda value...
  • atribute&=value - atributas, kurio reikšmė pasibaigia ...value
  • :link - paprasta/nelankyta nuoroda
  • :visited - lankyta nuoroda
  • :hover - užėjęs kursorius
  • :active - aktyvi paspaudimo momentu nuoroda
  • element:empty - tuščias elementas (be content)
  • ::marker - sąrašo elementų žymės
  • selector1::not(selector2) - parenka viską, apart nurodyto elemento
  • :first-child - parenka pirmą vaikinį elementą
  • :last-child - parenka paskutinį vaikinį elementą
  • :only-child - parenka vienintelį vaikinį elementą
  • :first-of-type - parenka pirmąjį nurodyto tipo elementą
  • :last-of-type - parenka paskutinįjį nurodyto tipo elementą
  • :only-of-type - parenka vientelį nurodyto tipo elementą
  • :nth-child(n) - parenka nurodytus vaikinius elementus
  • :nth-last-child(n) - nuo galo parenka nurodytus elemeentus
  • :nth-of-type(n) - parenka nurodytus elementus pagal tipą
  • :nth-last-of- type(n) - nuo galo parenka nurodytus elementus pagal tipą
  • ::before - sukuria kažkokį turinį prieš elementą
  • ::after - sukuria kažkokį turinį už elemento
  • input:checked - pažymėtus/parinktus radio/checkbox/option pasirinkimus
  • input:default - iš anksto parinktas pasirinkimas
  • input:disabled - input'as kuri turi disabled atributą
  • input:enabled - input'as, kuris neturi dissabled atributo
  • input:focus - input'as ant kurio yra paspausta / fokusuotas
  • input:in-range - input'as, kurio vertė yra nurodytose ribose
  • input:out-of-range - input'as, kurio vertė yra už nurodytų ribų
  • input:indeterminate - input'as, kurio reikšmė neapibrėžta
  • input:valid - input'as kurio vertė yra tinkama
  • input:invalid - input'as kurio vertė yra netinkama
  • input:required - input'as su required atributu
  • input:optional - input'as be required atributo
  • input:read-only - inputas su read-only atributu
  • input:read-write - input'as su nenurodytu read-only atributu
  • input / textarea::placeholder - įvesties laukelio placeholder atributo reikšmės parinkimas stilizavimui
  • ::first-letter - parenka elemento pirmąją raidę
  • ::first-line - parenka elemento pirmąją eilutę
  • ::selection - parenka vartotojo pažymėtą informaciją
  • :is()
  • :has()
  • :where()
