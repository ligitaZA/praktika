Responsive Design

Responsive design reaguoja į vartotojo įrenginį ir/arba jo plotį

Esant skirtingiems ekranų dydžiams galima pritaikyti skirtingus stilius naudojant @media query.

Break Points

  1. Phone(portrait)
  2. Table / landscape(Rotate) phone
  3. Laptop

Break Point width (px)

  1. <500px
  2. <678px
  3. <992px
  4. <1200px
  5. <1400px
  6. >1400px

Mobile First

  • Iš mažesnio ekrano dydžio yra lengvesnis perėjimas į didesnį(lengviau pridėti naujo turinio nei atimti)
  • Konsentracija į turinį dėl maž vietos kiekio.
  • Didesnis kiekis vartotojų naudoja mobiliąsias aplikacijas.
  • Google paieškos algoritmas geriau įvertina mobile-first dizainu kurtas svetaines.

Naudojant mobile first dizaino principą pradedame nuo mažesnių ekranų iki didesnių.
Tam naudojame media-query min-width:...px
